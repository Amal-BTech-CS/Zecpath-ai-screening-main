# AI Hiring System

# Overview

The AI Hiring System is a modular project designed to streamline and automate the recruitment process using Artificial Intelligence.
This project focuses on building a scalable architecture and clean development environment for future AI-based hiring features.

---

# Objective

To set up a professional AI development environment and design a well-structured, maintainable repository for building an intelligent hiring system.

---

# Tech Stack

* Python 3.x
* NumPy
* Pandas
* Scikit-learn
* FastAPI
* Pytest
